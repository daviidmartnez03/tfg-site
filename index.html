Descripción

Este trabajo presenta el desarrollo de una herramienta de software que automatiza la creación de pruebas para la certificación de software crítico en la industria aeronáutica. Los resultados obtenidos demuestran la viabilidad de unificar la validación de múltiples requisitos mediante la identificación de patrones de pruebas comunes bajo el principio de separar la lógica de validación de los datos concretos a evaluar. Esta creación se realiza mediante una sintaxis humanizada implementada a través de un DSL, contando con el apoyo de un asistente inteligente que actúa como soporte en la generación de los test automáticos. Este flujo permite optimizar el ciclo de verificación y superar las ineficiencias de métodos tradicionales basados en la creación manual de pruebas.
